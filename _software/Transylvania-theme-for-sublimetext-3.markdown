# Transylvania Theme for Sublime Text 3.
{% include post_header.html %}
This is a UI theme for Sublime Text 3 that goes well with the [Dracula syntax coloring scheme](https://draculatheme.com/).

It is work in progress and you can only obtain it from its [GitHub repo](https://github.com/silvercircle/ST_Transylvania). You need to install it manually, but once it's done, I'm going to submit it to package control.
