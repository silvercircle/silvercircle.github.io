The Pygments Style Class¶

This is what I am using for this site to highlight source code via pygments. This custom class is active when using the dark theme (via pygments_dark_style in the conf.py). Sphinx allows to customize the style depending on the CSS media query, so you can have different syntax highlighters based on whether the dark or light theme is active. For the light theme, I use the default highlight definitions, but it’s quite easy to customize them as well (see below)

How to use such a custom style?¶

  • Your style class must inherit from pygments.style.Style.

  • Copy this file to your sphinx site’s source directory (this is the directory that contains conf.py.)

  • Modify whatever color attribute you want.

  • In your conf.py make the path available for import like so:

1import os
2import sys
3sys.path.append(os.path.dirname(__file__))

  • In your conf.py use it with pygments_dark_style = "pyg_dark.Pyg_Dark". For a light theme use pygments_style instead.
