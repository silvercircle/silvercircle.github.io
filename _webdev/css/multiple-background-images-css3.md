# Multiple/layered background images with CSS3
{%include post_header.html%}
{%include defs.md%}
This is one of the more interesting features and allows for some cool effects, particularly when 
combined with transitions, gradients and transparency. A simple application of multiple, or more 
precisely, layered backgrounds can be seen in the header of this site. The background image is 
overlayed by a CSS gradient with gradually changing transparency and on top of everything, a semi 
transparent PNG image
