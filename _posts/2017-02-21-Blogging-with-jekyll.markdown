## Aren't static websites anti-social? 

Well, somehow they were considered anti-social before the Web 2.0 revolution 
with all the social media and dynamic sites took off many years ago. The main 
argument was that static web sites do not allow readers to interact with 
authors, but this was only half the truth. In fact, many authors and site 
publishers offered ways to interact with them, but the methods were different.
